# Akabot Studio

Akabot Studio là công cụ thiết kế quy trình tự động hóa trực quan, nơi bạn xây dựng, kiểm thử và xuất bản các workflow. Với canvas kéo-thả, thư viện hoạt động phong phú và công cụ debug tích hợp, cả lập trình viên lẫn người dùng không chuyên đều có thể tạo ra các bot mạnh mẽ mà không cần viết code lặp lại.

## Tính năng chính

- **Canvas thiết kế trực quan** — Kéo, thả và kết nối các activity để mô hình hóa bất kỳ quy trình nghiệp vụ nào.
- **Thư viện activity phong phú** — Hàng trăm activity tích hợp sẵn cho tự động hóa giao diện, xử lý file, thao tác dữ liệu, web service và nhiều hơn nữa.
- **Trình gỡ lỗi tích hợp** — Duyệt từng bước thực thi, kiểm tra giá trị biến và đặt breakpoint ngay trong giao diện thiết kế.
- **Quản lý phiên bản** — Commit và đẩy workflow lên repository Git để cộng tác nhóm.
- **Xuất bản lên Center** — Đóng gói và xuất bản workflow lên Akabot Center chỉ với một cú nhấp.
